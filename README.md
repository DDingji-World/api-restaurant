## Project rule
1. 변수명 -> camel case (e.g., ShopService)
2. 커밋은 최대한 구체적으로 하나의 역할만 의미하도록
   클래스 생성 단위, 메소드/로직 생성 단위, 변수명 생성/수정/삭제 단위
3. PR하면 다른 사람의 코드리뷰 후 Merge
4. package
   - controller
     - dto
   - applicationservice
   - domain
   - repository
   - global
     - config
     - exception
